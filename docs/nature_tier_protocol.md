## Phase 1: 数据扩展策略
**时间**: 1-2个月 | **成本**: $0

### 1.1 ChEMBL深度挖掘

**操作步骤**:
```python
# 目标: 获取5,000-10,000个DENV相关化合物
数据源:
- ChEMBL (主力): DENV NS3、NS5、NS2B靶点
- PubChem BioAssay: AID 1608, 1851等
- BindingDB: DENV相关条目
- 文献数据: 手工提取高质量数据

筛选标准:
- IC50/EC50/Ki有明确数值
- 分子量 150-600 Da
- 排除PAINS/非类药结构
```

**预期结果**: 
- 总化合物数: 8,000-12,000
- 高活性(pIC50>7): 2,000-3,000

**标准化流程**:
1. RDKit标准化（去盐、中性化）
2. 立体化学规范化
3. 互变异构体规范化
4. 去重（InChI Key匹配）
5. 活性值统一（转换为pIC50）

**数据划分**:
- 训练集: 70% (5,600-8,400)
- 验证集: 15% (1,200-1,800)

## Phase 4: 实验验证（关键）
**时间**: 3-6个月 | **成本**: $20,000-40,000

### 4.1 候选分子筛选

**计算筛选流程**:
```
Step 1: 初筛 (从10,000个生成分子)
- pIC50预测 > 7.5 (IC50 < 32 nM)
- QED > 0.7
- SA Score < 4.0
- 分子对接 ΔG < -8.5 kcal/mol
→ 保留500个

Step 2: 精筛
- 无PAINS/毒性警示
- 合成路线可行（< 8步）
- 新颖性检查（与已知药物Tanimoto < 0.85）
→ 保留50个

Step 3: 专家评审
- 3位药物化学家独立评分
- 选择10个合成候选
```

### 4.2 实验合作策略

**选项1: CRO外包**
- **公司**: WuXi AppTec, ChemPartner, Pharmaron
- **服务**: 合成 (5-10个) + 体外活性测试
- **成本**: $15,000-30,000
- **周期**: 3-4个月

**选项2: 学术合作**
- **寻找**: 有DENV研究背景的实验室
- **优势**: 成本低、可发表共同作者
- **劣势**: 周期较长（6-12个月）

**选项3: 自主合成（如有条件）**
- **设备需求**: Fume hood, 基础有机合成设备
- **人员**: 有机化学背景研究生
- **成本**: $5,000-10,000（试剂+耗材）

### 4.3 体外活性测试

**标准检测流程**:
```
1. DENV NS2B-NS3蛋白酶抑制实验
   - FRET法或荧光底物法
   - IC50测定（8个浓度点，3次重复）

2. 细胞抗病毒活性
   - Vero细胞DENV感染模型
   - EC50测定

3. 细胞毒性
   - MTT/CCK-8法
   - CC50测定
   - 选择性指数 SI = CC50/EC50

预期成功率: 2-3个分子IC50 < 100 nM
```

## Phase 5: 深度分析与创新

**分析内容**:

- R-group分解
- Matched Molecular Pairs (MMP)

**工具**:
- Schrödinger Maestro / GOLD
- OpenEye OEDocking
- PyMOL可视化

### 5.2 可解释AI分析

**方法**:
```python
1. SHAP值分析
   - 识别关键结构特征
   - 量化每个原子/基团贡献

2. 注意力机制可视化
   - GNN注意力热图
   - 药效团高亮

3. 对抗样本生成
   - 最小结构变化最大活性提升
   - 证明模型鲁棒性
```

### 5.3 Benchmark对比

**对比对象**:
- FDA批准的抗病毒药物（如有DENV数据）

- 临床候选物

**指标**:
- 活性 (IC50)
- 类药性 (QED, Lipinski)
- 效率指标 (LLE, LE)
